# Product Ops Triage

  A client-side customer feedback analysis tool built with React + Vite.

  Paste raw customer feedback (one item per line) and the app automatically classifies, clusters, and prioritizes it into actionable themes with:
  - Sentiment analysis (positive / neutral / negative)
  - Category classification (Bug, Feature Request, UX, Performance, Security, Data)
  - Severity scoring (critical / high / medium / low)
  - Source detection
  - Recommended actions per theme
  - Top 3 PRD ticket generation
  - Export to CSV, JSON, or Notion markdown

  No external APIs required — all analysis runs entirely in the browser.
